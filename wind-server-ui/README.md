# wind服务管理平台WEB UI

> 项目使用Vue3构建

## 安装
进入当前目录，执行安装命令
```shell
# 使用npm安装
npm install

# 使用淘宝npm源安装
npm install --registry=https://registry.npm.taobao.org

# 使用yarn安装
yarn

# 使用淘宝yarn源安装
yarn --registry=https://registry.npm.taobao.org
```

## 启动
```shell
# 使用npm启动
npm start

# 使用yarn启动
yarn start
```

## 构建
```shell
# 使用npm构建
npm run build

# 使用yarn构建
yarn build
```
